    • jQuery-ICE

    Initialize Callable Events

    Project maintained by EyeCode

    Set your template and let jquery-ICE call methods from raised events

    Prerequisite:

    • jQuery
    • jQuery-Class

    Install

    You can dowmload it from here or install via bower with the following package:

    "jquery-ice": "1.0.*"

    How To Use

    1) Include jquery-ice.js in the page (or use the minified version) with there dependency (jquery and jquery-class)

    2) Now play with the dom:

    <a data-ice="click|alert" data-ice-params="Hello World!">
    Click Here
</a>

    Exemple:
    Click Here

    Syntax

    <html data-ice="" [data-ice-params=""] [data-ice-event=""] [data-ice-callback=""]>

    html
    Type: String
    Any html tag is able to run this (a, div, button, etc...)

    data-ice
    Type: String
    Event following by the namespace/method to call, you just have to separate event from callable with a pipe.

    data-ice-params
    Type: Mixed
    String, Integer, Array or Object can be contained here as param to the called method, but becareful to write well defined object if used.

    data-ice-event
    Type: String
    Allows, directly from the dom, to register an event.

    data-ice-callback
    Type: String
    Allow callback to a method, you just have to call ice.callback(input, params) to raise the callback in method called previously. Useful to react after ajax call
